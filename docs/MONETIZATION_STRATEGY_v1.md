# QuizMaster 収益化戦略ドキュメント

## 1. エグゼクティブサマリー

QuizMasterは「科学的学習法 × AI自動生成 × 視覚記憶」という独自の価値提案を持つ学習アプリです。
本ドキュメントでは、ユーザー体験を最大化しながら持続可能な収益を生み出すための最適なプラットフォーム選択と販売戦略を提案します。

### 結論（推奨戦略）

| 優先度 | プラットフォーム | 理由 |
|--------|------------------|------|
| **1位** | **PWA + Webサービス** | 開発コスト最小、全デバイス対応、即時展開可能 |
| **2位** | **Chrome拡張** | Web学習との相性抜群、差別化要因になる |
| **3位** | ネイティブアプリ | 通知・習慣化に最強だが開発コスト高 |

## 3. プラットフォーム戦略

### 3.1 Phase 1: PWA + Webサービス（現状の延長）

**なぜPWAが最適か？**

```
現状のWebアプリ → PWA化
       ↓
┌────────────────────────────────────────┐
│ メリット                               │
├────────────────────────────────────────┤
│ ✅ 開発コストほぼゼロ（既存資産活用）    │
│ ✅ iOS/Android両対応                    │
│ ✅ ホーム画面追加でアプリライク          │
│ ✅ プッシュ通知対応（復習リマインダー）   │
│ ✅ オフライン対応（Service Worker）      │
│ ✅ App Store手数料なし（30%回避）        │
│ ✅ 即時アップデート反映                 │
└────────────────────────────────────────┘
```

**必要な実装:**
1. `manifest.json` の追加
2. Service Worker によるオフラインキャッシュ
3. Web Push API による復習リマインダー
4. インストールプロンプト UI

### 3.2 Phase 2: Chrome拡張（差別化の武器）

**なぜChrome拡張か？**

```
ユーザーの学習フロー:

  Webで記事を読む → 「これ覚えたい」 → QuizMaster拡張をクリック
       ↓                                      ↓
  ┌────────────────────────────────────────────────┐
  │ 1. ページのテキストを自動抽出                    │
  │ 2. ワンクリックでクイズ生成                     │
  │ 3. その場で復習 or 後で学習リストに追加         │
  └────────────────────────────────────────────────┘
```

**Chrome拡張のユニークバリュー:**

| 機能 | 効果 |
|------|------|
| **右クリックメニュー連携** | 選択テキストから即座にクイズ生成 |
| **ページ全体のクイズ化** | 技術記事、ニュース、論文を教材に |
| **学習リマインダー** | 新規タブに復習問題を表示 |
| **YouTube字幕対応** | 動画学習のアウトプット化 |

### 3.3 Phase 3: ネイティブアプリ（習慣化の完成形）

**いつネイティブ化すべきか？**

```
判断基準:
┌────────────────────────────────────────┐
│ ✅ MAU 10,000以上                      │
│ ✅ 有料課金率 5%以上                   │
│ ✅ PWAでの技術的限界に直面             │
│    - 通知の到達率が低い                │
│    - バックグラウンド同期が不安定      │
└────────────────────────────────────────┘
        ↓
  React Native または Flutter で開発
```

## 補足: なぜこの戦略か？

```
QuizMasterの本質的価値:
「どんな情報も30秒で学習可能な教材に変える」

この価値を最大化するプラットフォーム = Chrome拡張 + PWA

理由:
├── 学習は「インプット」と「アウトプット」のセット
├── Web上のインプット → すぐにアウトプット化できる
├── スマホでいつでも復習できる
└── ネイティブアプリより摩擦が少ない
```

**「見つけた情報をその場でクイズ化し、忘れる前に復習する」**

このサイクルを日常に組み込むことが、QuizMasterの勝ち筋です。
