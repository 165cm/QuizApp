# 📚 科学的勉強法クイズアプリ (QuizMaster)

**🚀 Deployed at: https://165cm.github.io/QuizApp/**

**科学的根拠に基づいた学習**で、記憶定着を最大化する次世代の学習Webアプリです。
PDF、テキスト、WebページからAIが自動でクイズを生成。さらに、生成AIによるイメージ画像が記憶のトリガーとなります。

[エンジニア向け技術ドキュメントはこちら (TECHNICAL_DOCS.md)](./TECHNICAL_DOCS.md)

---

## ✨ アプリの特徴

### 1. 🤖 AIによる自動クイズ生成 (マルチソース対応)
どんな素材からでも、30秒で学習用クイズが作れます。
- **📄 PDF**: 参考書や論文をアップロード
- **📝 テキスト**: ノートやメモを貼り付け
- **🌐 URL**: Web記事やWikipediaのリンク

### 2. 🎨 記憶に残る「画像生成」
文字だけの学習はもう終わり。Googleの最新AIモデルが、クイズの内容をイメージ化した画像を自動生成します。視覚情報とセットで覚えることで、記憶の定着率が飛躍的に向上します。
*   *v3.1で枠線を最小化、文字なし画像に最適化*

### 3. 📖 3段階解説カルーセル (v3 新機能)
解説を横スワイプで読めるスライド形式！
- **💡 ポイント**: 「実は…」で興味を引く
- **📖 解説**: 正解の理由を簡潔に
- **🚀 応用**: 「つまり〜」で実用的な知識に
- **⚠️ よくある間違い**: 多くの人がつまずくポイント

### 4. 🧠 科学的学習メソッドの実装
- **間隔反復 (Spaced Repetition)**: 忘却曲線に基づいて、最適なタイミングで復習を出題。
- **検索練習 (Retrieval Practice)**: 「思い出す」プロセスを強制し、脳の回路を強化。
- **適応型難易度**: あなたの理解度に合わせて、問題の難易度が「基礎」→「標準」→「応用」と変化。
- **復習リマインダー**: ホーム画面で復習待ち問題をお知らせ。

### 5. ☁️ クラウド同期 & クロスデバイス
Googleアカウントでログインすれば、データは自動的にクラウド (Supabase) にバックアップされます。
- PCで教材を作成し、移動中にスマホで復習する、といった使い方が可能です。
- オフラインでも一部機能は利用可能（同期はオンライン時に自動実行）。

### 6. 📤 簡単シェア & ウイルスループ
作った教材は友達や同僚とシェアできます。シェアされたクイズから新規ユーザーが作成する場合、自動的に**無料（テキストのみ）モード**で開始されるため、コストを気にせず拡散可能です。
- QRコードやリンクを送るだけ。受け取った人はアプリを開いてすぐに学習を開始できます。
- **PC最適化デザイン**: デスクトップでの学習時は、視線移動を最小限に抑える集中集中レイアウトに自動調整されます。

## 🚀 使い方ガイド

### ステップ1: 準備
1. **Googleログイン**: 右上の「ログイン」ボタンからログインします。（データのバックアップに推奨）
2. **APIキー設定**: 設定画面 (⚙️) から、OpenAI APIキーを入力します。

### ステップ2: 教材を作る
1. ナビゲーションバーの「教材生成 (➕)」をタップ。
2. PDF、テキスト、URLのいずれかを選んで入力。
3. 「生成」ボタンを押すと、AIが問題を分析・作成します。

### ステップ3: 学習する
1. 「ホーム (🏠)」画面で、今日の学習セットを開始。
2. 4択クイズに答えます。画像や解説をしっかり確認しましょう。
3. **ミニ復習**: 教材作成の待ち時間にも、過去の復習問題にチャレンジできます。

### ステップ4: 管理する
- **ライブラリ (📚)**: 作成したすべての教材を確認・編集できます。
- **レポート (�)**: 自分の得意・不得意や学習の継続記録（ストリーク）をグラフで確認できます。

## ⚠️ 注意事項

*   **API利用料**: OpenAI API (GPT-4o-mini) の利用料がかかります（非常に安価ですが、自己負担となります）。
*   **画像の保存期限**: 生成された画像は、ストレージ容量節約のため**30日後に自動削除**されます（クイズデータ自体は残ります）。
